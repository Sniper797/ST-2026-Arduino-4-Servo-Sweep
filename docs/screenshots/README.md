# Media assets

These files are used by the main [`README.md`](../../README.md) and the demo page
[`index.html`](../../index.html):

| File | What it is |
|---|---|
| `circuit.png` | The Tinkercad circuit — Arduino Uno + 4 servos on D8–D11. Hero image at the top of the README. |
| `demo.mp4` | Screen recording of the sketch running in the Tinkercad simulator (sweep → hold). Web-optimized (fast-start). |
| `demo-poster.jpg` | A still frame from the recording, used as the video thumbnail / poster. |

Want to add more? Drop the image in this folder and link it from `README.md`. PNG is best
for crisp screenshots; JPG is fine for photos and video stills.
